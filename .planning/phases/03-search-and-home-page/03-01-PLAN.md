**Milestone interface and getRecentMilestones():**
```typescript
export interface Milestone {
  bowlerID: number;
  bowlerName: string;
  slug: string;
  type: 'achieved' | 'approaching';
  milestone: string;
  current: number;
  threshold: number;
}
```
- Returns `Promise<Milestone[]>`
- Wrap in React.cache (used by both generateMetadata and page component)
- The query should find bowlers who recently crossed OR are approaching key thresholds
- Use a UNION ALL approach with two parts:

Part 1 — Recently achieved (career game count milestones): Find bowlers whose total game count (nightsBowled * 3 from scores where isPenalty=0) falls on a milestone threshold. Check thresholds: 50, 100, 150, 200, 250, 300, 400, 500 career games. A milestone is "recently achieved" if the bowler's total is within 9 games ABOVE the threshold (meaning they crossed it in the last ~3 bowling nights). Limit to bowlers from the most recent 2 seasons for relevance.

Part 2 — Approaching (within 5 games of next threshold): Find bowlers whose total game count is within 5 games BELOW a threshold. Same thresholds as above.

SQL approach:
```sql
-- Use a CTE to compute career game counts
WITH CareerGames AS (
  SELECT
    sc.bowlerID,
    b.bowlerName,
    b.slug,
    COUNT(sc.scoreID) * 3 AS totalGames
  FROM scores sc
  JOIN bowlers b ON sc.bowlerID = b.bowlerID
  WHERE sc.isPenalty = 0
  GROUP BY sc.bowlerID, b.bowlerName, b.slug
)
-- Recently achieved: totalGames between threshold and threshold + 9
SELECT bowlerID, bowlerName, slug,
  'achieved' AS type,
  CAST(threshold AS VARCHAR) + ' career games' AS milestone,
  totalGames AS current,
  threshold
FROM CareerGames
CROSS JOIN (VALUES (50),(100),(150),(200),(250),(300),(400),(500)) AS T(threshold)
WHERE totalGames BETWEEN threshold AND threshold + 9
UNION ALL
-- Approaching: totalGames between threshold - 5 and threshold - 1
SELECT bowlerID, bowlerName, slug,
  'approaching' AS type,
  CAST(threshold AS VARCHAR) + ' career games' AS milestone,
  totalGames AS current,
  threshold
FROM CareerGames
CROSS JOIN (VALUES (50),(100),(150),(200),(250),(300),(400),(500)) AS T(threshold)
WHERE totalGames BETWEEN threshold - 5 AND threshold - 1
ORDER BY type, threshold DESC
```
- Return up to 10 milestones total (add TOP 10 or slice in JS)
- Follow existing pattern: check env, try/catch, return [] on failure
