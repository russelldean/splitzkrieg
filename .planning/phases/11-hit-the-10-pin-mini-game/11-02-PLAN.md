    **src/components/game/GameEngine.ts** -- Matter.js physics world. DO NOT import Matter.Render (Pitfall 4). Use Matter.Engine, World, Bodies, Body, Events only.

    Class `GameEngine`:
    - Constructor creates Matter.Engine with `gravity: { x: 0, y: 0, scale: 0 }` (top-down view)
    - Creates ball body: `Bodies.circle(LANE_WIDTH/2, LANE_LENGTH - 50, BALL_RADIUS, { restitution: 0.3, friction: 0.05, density: 0.01, label: 'ball' })`
    - Creates pin body: `Bodies.circle(LANE_WIDTH/2, 60, PIN_RADIUS, { restitution: 0.5, friction: 0.1, density: 0.005, label: 'pin', isStatic: true })` (static until ball approaches)
    - Creates left/right gutter walls: static rectangles at x=0 and x=LANE_WIDTH
    - `update(delta: number)` calls `Matter.Engine.update(engine, delta)`
    - `launchBall(velocity: Vec2)` applies force to ball body via `Body.setVelocity`
    - `resetBall()` resets ball position to start, clears velocity
    - `getBallPosition(): Vec2` returns ball body position
    - `getPinPosition(): Vec2` returns pin body position
    - `getBallAngle(): number` returns ball body angle
    - `getPinAngle(): number` returns pin body angle
    - `isPinHit(): boolean` checks if pin has moved significantly from start position
    - `isBallOutOfBounds(): boolean` checks if ball has passed pin Y or entered gutter
    - `destroy()` clears the Matter.js world
