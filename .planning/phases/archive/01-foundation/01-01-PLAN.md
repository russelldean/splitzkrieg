  ```css
    --color-cream: #FAF7F2;

    --font-body: var(--font-inter);
    --color-background: var(--color-cream);
    --color-foreground: var(--color-navy);
  }
  ```

  Add base styles after the theme block: `body { @apply bg-cream text-navy font-body antialiased; }`. Add utility classes for the editorial feel: `.heading { @apply font-heading; }`.
